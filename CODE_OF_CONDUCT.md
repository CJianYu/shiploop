# Code of Conduct

We are committed to a respectful, harassment-free community. Be constructive, assume good intent,
criticize ideas rather than people, and respect privacy and attribution. Maintainers may remove
content or participation that undermines a safe and productive project environment.

Report conduct concerns privately to the project maintainers through the repository's published
contact channel. Until one is published, use a private GitHub security advisory rather than a public
issue.
